# mac-dev-cleanup-helper

I made this because my Mac had too many random dev installs and old versions
and I couldn’t remember what was actually being used.

This is a tiny Python script that shows:

- programming tools on your PATH (with real paths + versions)
- common dev folders on macOS

It helps you decide what to clean.

It does NOT delete anything.

---

## How to run

```bash
python3 dev_stuff_finder.py

```
